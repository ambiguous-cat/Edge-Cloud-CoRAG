# 📚 知识库管理系统

一个功能完整的知识库管理前端界面，提供文档的增删查功能。

## 🚀 快速开始

```bash
# 1. 启动API服务器
python api_server.py

# 2. 启动管理界面
cd 知识库前端
python advanced_manager.py
```

## 📋 功能特性

### 🔍 文档搜索
- 关键词搜索知识库文档
- 可调节返回结果数量
- 显示相似度评分
- 表格化结果展示

### 📋 文档管理
- 查看所有文档列表
- 获取文档详细信息
- 显示文档统计信息
- 支持文档ID快速查找

### ➕ 文档添加
- **文件上传**: 支持TXT格式文件上传
- **JSON添加**: 直接输入结构化JSON内容
- 自动生成嵌入向量
- 实时反馈添加结果

### 🗑️ 文档删除
- 根据文档ID删除文档
- 自动清理相关文档块
- 安全确认机制
- 删除结果反馈

### 📊 统计信息
- 数据库整体统计
- 最近添加文档
- 文档和块数量统计

## 🖥️ 界面说明

### 高级管理界面 (`advanced_manager.py`)
- 标签页式布局，功能分区清晰
- 完整的增删查管理功能
- 详细的统计和监控信息
- 现代化渐变色设计

## 📝 JSON格式要求

添加JSON文档时，请使用以下格式：

```json
[
  {
    "text": "这是第一段内容",
    "section": "章节1",
    "block_id": 1
  },
  {
    "text": "这是第二段内容", 
    "section": "章节2",
    "block_id": 2
  }
]
```

### 字段说明
- `text`: 文档块内容（必需）
- `section`: 章节名称（可选）
- `block_id`: 块ID（可选）

## ⚠️ 注意事项

1. **API服务器**: 确保主目录的 `api_server.py` 正在运行
2. **数据库**: 使用主目录的 `local_knowledge.db` 数据库
3. **FAISS索引**: 删除文档后需要手动重建FAISS索引
4. **文件格式**: 目前仅支持TXT格式文件上传
5. **端口冲突**: 默认使用7860端口，如有冲突请修改配置

## 🔗 API端点

系统使用以下API端点：

- `POST /search` - 文档搜索
- `POST /add_document` - 添加文档文件
- `POST /add_json_document` - 添加JSON文档
- `GET /docs` - API文档和健康检查

## 🐛 故障排除

### API服务器连接失败
1. 检查 `api_server.py` 是否运行
2. 确认端口8005未被占用
3. 检查防火墙设置

### 文件上传失败
1. 确认文件格式为TXT
2. 检查文件路径和权限
3. 确认文件编码为UTF-8

### JSON格式错误
1. 使用JSON验证工具检查格式
2. 确保是数组格式 `[...]`
3. 检查字段名称是否正确

## 📞 技术支持

如遇问题，请检查：
1. 控制台错误信息
2. API服务器日志
3. 数据库文件是否存在
4. 网络连接状态
